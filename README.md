# 🧮 Omega Calculator 11.0 
The graphical edition of Omega Calculator 10.0, rebuilt with a modular, object-oriented architecture and a modern GUI powered by Kivy. Includes three optimized editions for mobile, tablet, and desktop.

For the original console-based modular version, see [Omega Calculator 10.0](https://github.com/AmirmahdiGhorbani2000/Omega_Calculator_10.0).

## Editions

Three editions are included, each tailored to a specific screen size:

- `Mobile Edition.py` — Mobile (360×640)
- `Tablet Edition.py` — Tablet (720×1280)
- `Desktop Edition.py` — Desktop (1024×768)

---

## Features

- 69 physics formulas across 8 categories
- 118 elements of the periodic table with search
- Polynomial solver for degrees 1 through 10
- Matrix operations: addition, multiplication, determinant, and inverse
- Numerical calculus: derivative and Simpson integration
- Statistics: mean, median, mode, and distributions
- Geometry: area, perimeter, volume, and trigonometry
- Unit converter for length, temperature, and weight
- Clean graphical interface with 9 tabs
